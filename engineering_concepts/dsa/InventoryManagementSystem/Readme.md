###Suitable Data Structures
1. ArrayList-easy to implement but searching,updating and deleting products require linear traversal.
2. HashMap–stores products using a unique product ID as the key,providing fast access, insertion,update and deletion.

Choosen Data Structure-*HashMap*(optimized)

###Time Complexity
add-O(1)
update-O(1)
delete-O(1)
search-O(1)
