##Time Complexity Comparison
###Linear Search:
Best Case-O(1)
Average Case-O(n)
WorstCase-O(n)

###Binary Search:
Best Case-O(1)
Average Case-O(log n)
WorstCase-O(log n)

*Binary Search* is more suitable for an e-commerce platform because:
1. It is much faster for large and sorted datasets whereas linear search is better for small datasets.
2. It reduces the search space by half in every iteration.
3. It performs in O(log n) time whereas linear search performs in O(n).
