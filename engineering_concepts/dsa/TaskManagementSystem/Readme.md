###Singly Linked List
It consists of nodes where each node stores data and a reference to the next node. Traversal is possible only in the forward direction.

###Doubly Linked List
It consists of nodes where each node contains data, a reference to the next node, and a reference to the previous node. It allows traversal in both forward and backward directions.

###Time Complexity
add-O(n)
search-O(n)
traverse-O(n)
delete-O(n)

###Advantages of LL over Arrays
1. Dynamic size 
2. Easy insertion and deletion without shifting elements
